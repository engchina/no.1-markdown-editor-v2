# AGENTS.md

## Mission

Build the world's best Markdown editor.

## Non-Negotiable Product Principles

1. Use `ui-ux-pro-max` for any task involving UI or UX.
2. Study `Typora`, `Obsidian`, `VS Code Markdown`, `Bear`, `iA Writer`, and `Notion`. Identify what each product does well, then adapt those principles to this product's direction instead of copying them blindly.
3. Treat `Typora Markdown Reference` as the primary reference for Markdown behavior and compatibility decisions.
4. Support `Windows`, `macOS`, and `Linux`.
5. Support `Japanese`, `English`, and `Chinese`.
6. Choose the technology stack that is best suited to achieve the goals above, with strong bias toward product quality, maintainability, performance, and cross-platform consistency.

## Decision Standard

When there is a tradeoff, prefer the option that most directly advances the mission of building the world's best Markdown editor.
